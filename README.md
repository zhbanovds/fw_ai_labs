# ML Frameworks Labs

Репозиторий с лабораторными работами по дисциплине, связанной с исследованием алгоритмов машинного обучения на задачах классификации и регрессии.

## Структура репозитория

- `lab1_fw_ai.ipynb` — исследование алгоритма KNN
- `lab2_fw_ai.ipynb` — исследование логистической и линейной регрессии
- `lab3_fw_ai.ipynb` — исследование решающего дерева
- `lab4_fw_ai.ipynb` — исследование случайного леса
- `lab5_fw_ai.ipynb` — исследование градиентного бустинга
- `data/` — используемые наборы данных
- `requirements.txt` — зависимости проекта

## Используемые датасеты

### 1. Breast Cancer Wisconsin Diagnostic
Набор данных для задачи бинарной классификации.  
Целевая переменная показывает тип диагноза, а признаки описывают характеристики опухоли.

Используется в лабораторных работах для:
- KNN-классификации
- логистической регрессии
- решающего дерева
- случайного леса
- градиентного бустинга

### 2. Concrete Compressive Strength
Набор данных для задачи регрессии.  
Целевая переменная отражает прочность бетона на сжатие, а признаки описывают состав бетонной смеси.

Используется в лабораторных работах для:
- KNN-регрессии
- линейной регрессии
- решающего дерева
- случайного леса
- градиентного бустинга

## Содержание лабораторных работ

### Лабораторная работа 1
Исследование алгоритма KNN для задач классификации и регрессии:
- построение baseline
- выбор метрик качества
- проверка гипотез по улучшению качества
- построение improved baseline
- собственная реализация алгоритма
- сравнение собственной реализации с `scikit-learn`

### Лабораторная работа 2
Исследование логистической и линейной регрессии:
- baseline-модели
- анализ влияния масштабирования
- исследование регуляризации
- подбор гиперпараметров
- собственная реализация логистической и линейной регрессии
- сравнение результатов

### Лабораторная работа 3
Исследование решающего дерева:
- baseline-модели
- анализ влияния глубины дерева и параметров разбиения
- подбор гиперпараметров
- анализ важности признаков
- визуализация структуры дерева
- собственная реализация алгоритма
- сравнение с библиотечной реализацией

### Лабораторная работа 4
Исследование случайного леса:
- baseline-модели

- подбор гиперпараметров
- анализ важности признаков
- собственная реализация ансамбля деревьев
- сравнение с библиотечной реализацией

### Лабораторная работа 5
Исследование градиентного бустинга:
- baseline-модели

- исследование процесса обучения
- собственная реализация бустинга
